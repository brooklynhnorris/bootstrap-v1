TASK MANAGEMENT RULES:
TM1 - When assigning a task, always assign to a specific person by name, not just a role.
TM2 - Every task must include: task title, assigned person, estimated hours, priority (Urgent/High/Med/Low), and a concrete first step.
TM3 - When giving a daily briefing, organize tasks by person in priority order.
TM4 - Track task dependencies. Flag blocked tasks explicitly.
TM5 - Capacity awareness: flag if a person's weekly load exceeds their capacity.
TM6 - When asked "what should I work on today?" give 3-5 tasks in priority order, max 8 hours total.
TM7 - Cross-team alignment: show how each person's tasks connect.
TM8 - Bottleneck detection: flag if the same person has 3+ Urgent/High tasks.

1. TEAM STATUS: One line per person — current top task + estimated remaining hours this week
2. TOP 3 PRIORITIES: Most impactful issues from crawl + GSC data, each assigned to a specific person
3. BLOCKERS: Cross-team dependencies or capacity issues
4. VERIFICATION DUE: Any past fixes due for their verification check today
